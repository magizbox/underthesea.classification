 Sun tung ra Java Studio Creator 2.0
Java Studio Creator 2 được thiết kế với nhiều cải tiến độc đáo, có nhiều công cụ tạo ứng dụng Java cho web, bao gồm một thư viện đầy đủ mọi thành phần thiết kế tương thích hoàn hảo với các cấu trúc và công nghệ ngôn ngữ lập trình mới nhất như Asynchronous JavaScript và XML (AJAX). 
Java Studio Creator 2 cũng đồng tương thích với các công nghệ lập trình Netbaeans IDE 4.1; JavaServer Faces Technology thường được sử dụng trong các ứng dụng AJAX tương thích với chuẩn JSR-168, được cho là hiện đại nhất hiện nay. Nó tích hợp hoàn hảo giữa các cổng Java và cổng Web, có khả năng gia tăng kiểm soát các ứng dụng đồ họa, CSS trong các công cụ tạo trang HTML. 
Sun đã tung ra phiên bản đầu tiên của Java Studio Creator trong năm 2004. Sun có lẽ cũng đã “bắt chước” một số chiêu thức thiết kế công cụ sao cho thật dễ sử dụng từ Visual Studio.Net của Microsoft. Phiên bản đời thứ hai của Java Studio Creator này cũng đã chứng tỏ được sức sống cực kỳ mạnh mẽ của ngôn ngữ Java trên web và nó cũng là công cụ dùng để thu hút thêm nhiều chuyên gia lập trình chuyển sang sử dụng Java làm đối trọng với hệ ngôn ngữ Visual của Microsoft. 
Công cụ này hiện đang được phân phối miễn phí cho tất cả mọi lập trình viên trên toàn cầu đã đăng ký tại mạng Sun Developer Network. Người dùng quan tâm có thể vào đây để tìm hiểu và tải về sử dụng. Hiện Java Studio Creator 2.0 chạy hoàn hảo trên Windows, Mac OS X and, Linux và Solaris
